# Diet-dataset
This dataset is synthetic and uncleaned, designed to simulate realistic user data for diet recommendation purposes. It contains 3030 examples, each representing a unique individual with body measurements, body composition, workout habits, and fitness goals.




Feature Overview
Age: Age of the user in years

Gender: Encoded gender (e.g., 0 = Female, 1 = Male)

Weight (kg): Body weight in kilograms

Height (m): Height in meters

BMI: Body Mass Index calculated as Weight / (HeightÂ²)

Fat%: Body fat percentage

Workout_Frequency: Number of workout days per week

Goal_Label: Fitness goal:  Maintain, Muscle Gain, Weight Loss

Diet_Type: Target variable: Balanced, High-Protein, Low-Carb
